# 💳 Customer Segmentation App

An AI-powered **Customer Segmentation Dashboard** built using Machine Learning (KMeans Clustering) and Streamlit.

---

## 🚀 Live App
👉 https://customer-segmentation-app.streamlit.app

---

## 📊 Features

- Upload customer dataset (CSV)
- Automatic clustering using KMeans
- PCA visualization of customer groups
- Business-friendly insights
- Download segmented data

---

## 🧠 Customer Segments

💎 VIP Customers → High value customers  
💰 High Spenders → Large transactions  
⚠️ Cash Users → Frequent cash advance usage  
❄️ Low Activity → Minimal engagement  

---
